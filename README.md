# Tempature Control App

Control the background based on the temperature specified on the screen.

_note: not for real world use._

## Features

-   Real-time Input & Background Updates.
-   Live Preview
-   Mobile Accessible

## Deployment

Clone or Download

```bash
git clone https://github.com/ttommie/temperature-control-app.git
```

Install all node modules

```bash
npm install
```

Run the react server

```bash
npm start
```
